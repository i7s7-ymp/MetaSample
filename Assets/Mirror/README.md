# 構造

| ディレクトリ | 内容 | 役割 |
|---|---|---|
| Authenticators | .cs, .asmdef | ネットワーク接続時の認証方式（Basic認証、デバイスID認証など）を実装するスクリプトが含まれています。 |
| CompilerSymbols | .cs, .asmdef | コンパイラシンボルを管理し、特定の条件下でコードをコンパイルに含めたり除外したりするためのスクリプトが含まれています。 |
| Components | .cs, .asmdef | ネットワーク機能を実現するための主要なコンポーネント（NetworkAnimator, NetworkLobbyManagerなど）が含まれています。 |
| Editor | .cs, .asmdef | Unityエディタの拡張機能。Mirror関連コンポーネントのインスペクタ表示をカスタマイズしたり、エディタ上での操作を補助したりするスクリプトが含まれています。 |
| Examples | .unity, .cs, .prefab, .asmdef | Mirrorの各機能を使ったサンプルプロジェクトが含まれています。 |
| Icon | .png | Mirrorのアイコン画像が含まれています。 |
| Plugins | .dll | サードパーティ製のライブラリ（Mono.Cecilなど）が含まれています。Mirrorが内部的に利用します。 |
| Runtime | .cs, .asmdef | Mirrorのコアとなるランタイムスクリプトが含まれています。ネットワーク通信、オブジェクトの同期、メッセージングなど、基本的なネットワーク機能が実装されています。 |

# ファイル拡張子の説明

| 拡張子 | 説明 |
|---|---|
| .asmdef | Assembly Definitionファイル。スクリプトをアセンブリ（DLL）に分割して管理するための設定ファイルです。コンパイル時間を短縮し、依存関係を整理するのに役立ちます。 |
| .cs | C#スクリプトファイル。Unityのコンポーネントとしてゲームオブジェクトにアタッチし、動作を記述するために使用されます。 |
| .dll | 動的リンクライブラリ。コンパイル済みのコードを含み、他のプログラムから利用されます。Mirrorでは主に`Plugins`ディレクトリに配置されます。 |
| .png | 画像ファイル形式。主にアイコンやUI要素として使用されます。 |
| .txt | テキストファイル。ドキュメントや簡単なメモなどに使用されます。 |
| .unity | シーンファイル。ゲーム内のレベルや画面構成を保存します。ゲームオブジェクト、ライト、カメラなどが含まれます。 |
| .prefab | プレハブファイル。設定済みのゲームオブジェクトを再利用可能なアセットとして保存したものです。 |
| .meta | Unityが生成するメタデータファイル。アセットのインポート設定やGUID（一意なID）などを保持します。全てのアセットファイルに1対1で対応して作成されます。 |
| LICENSE | 拡張子なしのライセンスファイル。 |
